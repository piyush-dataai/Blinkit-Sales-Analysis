# Blinkit Sales Analysis Dashboard

## Project Overview

This Power BI dashboard analyzes Blinkit's grocery sales data to uncover insights related to sales performance, outlet characteristics, customer ratings, and product categories.

## Key Business Insights

- Tier 3 outlets generated the highest sales.
- Medium-sized outlets performed best overall.
- Fruits & Vegetables and Snack Foods were the top-selling categories.
- Average customer rating remained consistent across outlet types.
